# Jainil Parekh Portfolio

Personal portfolio website built with Vite, React, TypeScript, and Tailwind CSS.

The site showcases:

- Professional summary and hero section
- Featured AI, ML, backend, and analytics projects
- Experience, education, certifications, and technical skills
- Contact section for direct outreach

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Local Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Project Structure

```text
project/
├── public/        # static assets such as profile image and project images
├── src/
│   ├── components/
│   ├── App.tsx    # main portfolio page
│   └── index.css  # global styles
├── index.html
└── package.json
```

## Customization

Most portfolio content is managed inside:

- `src/App.tsx`

Update these sections there:

- hero content
- social links
- projects
- experience
- education
- certifications
- skills

## Deployment

This project is ready to deploy on platforms such as:

- Vercel
- Netlify
- GitHub Pages

For Vercel, the default Vite settings are typically enough:

- Build command: `npm run build`
- Output directory: `dist`
